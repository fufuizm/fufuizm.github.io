Doğrulama / Çalışmalarımı Nasıl Kanıtlıyorum Verification / How I Validate My Work
  • ✓ Restore testi: Backup'ları düzenli olarak geri yükleyerek bütünlüğünü doğrulama Restore test: Periodically restore backups to verify integrity before relying on them
  • ✓ Paket yakalama: Her ağ yapılandırma değişikliğinin ardından tcpdump ile trafik analizi Packet capture: tcpdump/Wireshark traffic analysis after every network config change
  • ✓ Log korelasyonu: Servis olaylarını syslog, auth.log ve uygulama loglarıyla çapraz doğrulama Log correlation: Cross-validating service events against syslog, auth.log, and app logs
  • ✓ Sertleştirme kontrol listesi: SSH anahtar kimlik doğrulaması, devre dışı parola girişi, ufw varsayılan reddet, root oturumu kapalı Hardening checklist: SSH key auth, disabled password login, ufw default-deny, root login off
  • ✓ Port tarama doğrulaması: Firewall kural değişikliklerinin ardından nmap ile saldırı yüzeyi kontrolü Port-scan validation: nmap scan against all VMs after firewall rule changes to confirm attack surface
  • ✓ Kaos testi: Servis kesintisi simülasyonu ve alert / otomatik kurtarma süresini ölçme Chaos test: Simulate service outage and measure alert-to-recovery time against RTO targets

# wireguard-setup.yml — Ansible Playbook
---
- name: Configure WireGuard VPN Server
  hosts: vpn_host
  become: yes
  vars:
    wg_interface: wg0
    wg_port: 51820
    wg_network: "10.8.0.0/24"
    server_ip: "10.8.0.1"

  tasks:
    - name: Install WireGuard
      apt:
        name: wireguard
        state: present
        update_cache: yes

    - name: Generate server private key
      shell: wg genkey
      register: server_private_key

    - name: Deploy wg0 config
      template:
        src: wg0.conf.j2
        dest: /etc/wireguard/wg0.conf
        mode: '0600'

    - name: Enable and start WireGuard
      systemd:
        name: "wg-quick@{{ wg_interface }}"
        enabled: yes
        state: started

    - name: Configure ufw rules
      ufw:
        rule: allow
        port: "{{ wg_port }}"
        proto: udp
